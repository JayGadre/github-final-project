# github-final-project

A straightforward tool to calculate simple interest for a given principal amount, annual interest rate, and time period.

## Formula

The calculator uses the standard simple interest formula:

$$SI = \frac{P \times R \times T}{100}$$

Where:
* **P** = Principal amount (initial investment or loan amount)
* **R** = Annual interest rate (in percentage)
* **T** = Time (in years)

## Features
* Calculates total simple interest accrued.
* Computes the final total amount (Principal + Interest).
* Lightweight and easy to integrate.
